# instagram_chat

Instagram bot - avtomatik xabar yuborish va comment javoblash tizimi.

## O'rnatish

```bash
pip install -r requirements.txt
```

## Ishga tushirish

```bash
python app.py
```

## Xususiyatlar

- Instagram akkauntlarini boshqarish
- Kalit so'zlar bo'yicha avtomatik javob
- Comment va DM javoblash
- Web interfeys orqali boshqarish
